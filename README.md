# sales
    ## install
        * create virtual enviroment
            python -m venv venv
        * activate virtual enviroment ( Windows )
            cmd .\venv\Scripts\activate.bat
            powershell:  .\venv\Scripts\activate.ps1 
        * install dependecies
            pip install -r .\requirements.txt
    
    ## run server
        shell version
            python ./src/main.py

        windows version
            pythonw ./src/main.py

    ## open application
        in web browser url: http://localhost:8080
